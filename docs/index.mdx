## üöÄ Quick Start

### Using Docker (Recommended)

**Prerequisites:** [Docker](https://docs.docker.com/get-docker/)

```bash
# Clone the repository
git clone https://github.com/losol/eventuras.git
cd eventuras

# Build and run the application
docker-compose up
```

The application will be available at `http://localhost:5100`.

## üèóÔ∏è Architecture

Eventuras is built as a **monorepo** with:

- **Backend:** .NET Core API with PostgreSQL
- **Frontend:** Next.js applications with React and TypeScript
- **Libraries:** Shared TypeScript/React libraries
- **Infrastructure:** Docker, Turborepo, npm workspaces

For a complete architectural overview, see [Architecture Overview](./developer/Architecture_overview.md).
