### 🧱 Features

Add TreeView, TableOfContents, and ThreeColumnLayout components

New reusable components for documentation sites and similar layouts:

- TreeView: hierarchical collapsible navigation with active-path
  highlighting, LinkComponent prop for SPA routers, and full ARIA support
- TableOfContents: sticky sidebar with IntersectionObserver scroll-spy,
  aria-current on active heading, and primary color tokens
- ThreeColumnLayout: responsive three-column grid (left sidebar, content,
  right sidebar) with sticky side columns and labeled landmarks
