# App notifications component

The easiest way to show notifications in your app is to use the useAppNotifications hook. It provides a function to add notifications to the store and uses the component to render them.

```tsx
const { addAppNotification } = useAppNotifications();
```

## Add notifications

```tsx
addAppNotification({
  id: Date.now(),
  message: 'This is a success notification',
  type: AppNotificationType.SUCCESS,
});
```
